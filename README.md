## サービス概要
サウナ好きがととのい体験・体感したこだわりを記録できるアプリケーションです。
行ったサウナ施設の情報や感想、満足度を記録できます。
自分だけのサウナ履歴コレクションとしても、他ユーザーとの情報共有にも使えます。

## このサービスへの思い・作りたい理由
自分自身サウナに行くのが好きで、「どの施設が良かったか」「何度目でどんな整い方をしたか」を記録したいと思ったのがきっかけです。
今までは時間が経つとどんなサウナだったか忘れてしまうこともあったので、記録として残せて後から見返せるアプリケーションがあれば良いなと思いました。

## ユーザー層について
メイン層はサウナが趣味の20〜40代男性で記録・コレクション欲を掻き立てられるアプリケーションを目指します。
そのほか、サウナに興味がある初心者層にもアプローチしたいです。

## サービスの利用イメージ
ユーザーはサウナに行った後に施設名や温度・ととのい度と感想など登録します。
これにより自分の「サウナ履歴帳」として活用でき、どこの施設が自分に合っていたかを振り返ることができます。
他のユーザーの投稿を見て、新しい施設を発見する楽しみも得られます。

## サービスの差別化ポイント・推しポイント
既存のサウナ情報アプリは施設検索や簡単なレビュー重視が多いですが、本サービスは「自分の体験とこだわりを記録する」ことに重点を置いています。
その時に感じた小さなこだわりでも、例えばタイトルと本文を自由に設定できる欄へ「備品のメーカー」「シャワーの水圧」など記録を残しておける独自性は推しポイントです。

## 使用する技術スタック
- 使用するフレームワーク
  Ruby on Rails 7.x
- データベース
  MySQL 8.x
- デプロイ先
  Render
- 使用予定のGem
  Devise：ユーザー認証機能（ログイン・ログアウト・登録）を実装。
　Active Storage：画像アップロード機能（サウナ施設写真など）に使用。
　Ransack（または簡易検索機能実装）：投稿の検索・ソート機能に利用予定。
　Chartkick + Groupdate：整い度の統計グラフ表示を実装予定。
　Tailwind CSS：デザインを効率的に整えるためのCSSフレームワーク。
　dotenv-rails：APIキーや環境変数を安全に管理。
